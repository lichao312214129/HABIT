# HABIT 文档

这是 HABIT 项目的官方文档。

## 构建文档

### 安装依赖

```bash
pip install -r requirements.txt
```

### 构建 HTML 文档

```bash
make html
```

或 Windows:

```cmd
make.bat html
```

### 实时预览

```bash
make livehtml
```

## 文档结构

- **getting_started/**: 快速开始指南
- **user_guide/**: 用户指南
- **api/**: API 文档
- **algorithms/**: 算法文档
- **tutorials/**: 教程
- **development/**: 开发者指南
