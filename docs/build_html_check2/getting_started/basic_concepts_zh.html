基本概念

生境（Habitat） ：肿瘤 ROI 内影像特征相近的亚区；不同颜色/标签代表不同亚区，需结合各序列信号解读。

典型流程 ：预处理 → 生境分割 → 特征提取 →（可选）机器学习。

聚类策略（生境分割） ：

  • Two-Step （推荐）：先个体超像素，再群体生境

  • One-Step ：单肿瘤内直接分生境

  • Direct Pooling ：全体体素一起聚类

操作与 demo 命令见 完整 Demo 教程；参数见 配置参考。
