Installation
============

Prerequisites
-------------

HABIT requires Python 3.8 or higher.

Install from Source
--------------------

.. code-block:: bash

   git clone https://github.com/your-repo/habit_project.git
   cd habit_project
   pip install -e .

Install Dependencies
--------------------

.. code-block:: bash

   pip install -r requirements.txt

Verification
------------

Verify the installation:

.. code-block:: bash

   habit --version
