class habit.core.habitat_analysis.clustering.slic_clustering.SLICClustering(n_clusters: int | None = None, random_state: int = 0, compactness: float = 0.1, max_iter: int = 10, sigma: float = 0.0, enforce_connectivity: bool = True, n_init: int = 10, spacing: Tuple[float, float, float] | None = None, **kwargs: Any)[源代码]

SLIC-based clustering for voxel-level habitat analysis.

备注

  • predict is only valid for the same subject after fit because SLIC is not a parametric model for unseen samples.

  • Output labels are returned as 0-indexed values to match the current HABIT convention (manager adds +1 later).

__init__(n_clusters: int | None = None, random_state: int = 0, compactness: float = 0.1, max_iter: int = 10, sigma: float = 0.0, enforce_connectivity: bool = True, n_init: int = 10, spacing: Tuple[float, float, float] | None = None, **kwargs: Any) → None[源代码]

Initialize SLIC clustering.

参数:

  • n_clusters -- Target number of supervoxels/habitats.

  • random_state -- Random seed for reproducibility.

  • compactness -- Trade-off between appearance and spatial proximity.

  • max_iter -- Maximum number of iterations for SLIC.

  • sigma -- Gaussian smoothing width before segmentation.

  • enforce_connectivity -- Whether to enforce connected supervoxels.

  • n_init -- Number of initializations for KMeans in optimal-k search.

  • spacing -- Optional voxel spacing (z, y, x) for anisotropic data.

  • **kwargs -- Reserved for compatibility.

find_optimal_clusters(X: ndarray, min_clusters: int = 2, max_clusters: int = 10, methods: List[str] | str | None = None, show_progress: bool = True, spatial_coords: ndarray | None = None, spacing: Tuple[float, float, float] | None = None, logger: Logger | None = None) → Tuple[int, Dict[str, List[float]]][源代码]

Find optimal cluster number using spatially regularized features.

This method evaluates K values on a SLIC-style feature embedding (appearance + compactness-weighted coordinates). It is used only for model selection in one-step mode.

参数:

  • X -- Feature matrix with shape (n_samples, n_features).

  • min_clusters -- Minimum cluster number to evaluate.

  • max_clusters -- Maximum cluster number to evaluate.

  • methods -- Validation methods; supports silhouette, calinski_harabasz, davies_bouldin, inertia, kneedle.

  • show_progress -- Whether to emit search progress logs.

  • spatial_coords -- Optional voxel coordinates for spatial regularization.

  • spacing -- Optional voxel spacing (z, y, x). Reserved for API compatibility.

  • logger -- Optional logger for per-k search progress.

返回:

Best cluster number and score dict.

返回类型:

Tuple[int, Dict[str, List[float]]]
