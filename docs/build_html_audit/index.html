HABIT: Habitat Analysis: Biomedical Imaging Toolkit

HABIT**用于肿瘤 生境（Habitat）**分析：从医学影像中划分肿瘤亚区并提取定量特征，可选构建预测模型。

  1. 安装指南 — 安装环境

  2. 完整 Demo 教程 — 跑通 demo（5 步命令）

  3. 用户指南 — 各步骤最短用法

  4. 配置参考 — YAML 参数全集（改配置时查阅）

流程 ：预处理 → 生境分割 → 特征提取 → 机器学习（可选）
