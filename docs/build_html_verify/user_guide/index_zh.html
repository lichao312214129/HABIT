用户指南

面向临床用户的 最短路径 ：先完成 getting_started/quickstart_zh 跑通 demo，再按需打开下面各步。各命令的 YAML 字段、默认值 见 配置参考。

一、工作流程

  1. 预处理 — 图像预处理

  2. 生境分割 — 生境分割

  3. 特征提取 — 生境特征提取

  4. 机器学习 — 机器学习建模

  5. 模型对比 （可选）— 模型对比

  • 图像预处理
  • 生境分割
  • 生境特征提取
  • 机器学习建模
  • 模型对比

二、通用约定

  • 每次运行前：Windows 用 Anaconda Powershell Prompt ，conda activate habit，cd 进入项目根（示例 cd "D:\HABIT-main"；路径见 安装指南）。

  • Demo 影像根目录： demo_data/preprocessed/processed_images/``（被试 ``subj001 / subj002，序列 delay2 / delay3 / delay5）。

  • 修改 config/ 下 YAML 中的路径与参数；勿在本文重复罗列参数。

  • ROI 请用 ITK-SNAP 或 3D Slicer 勾画，保存为 NIfTI；数据目录结构见 data_structure_zh。
